# FCIFNet
This project provides the code and results for 'FCIFNet:Feature correction and Iterative Fusion Network for RGB-D salient object detection'
## Requirements
- python 3.9  
- pytorch 2.0.1  
- torchvision 0.15.2  
- opencv-python 4.11.0  
- numpy 1.22.4  
- timm 0.6.5
